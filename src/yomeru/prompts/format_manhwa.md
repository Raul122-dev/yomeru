# Manhwa (Korean) — Reading format

## Reading direction: LEFT to RIGHT, TOP to BOTTOM

Standard western reading order. Most manhwa is formatted as vertical scroll (webtoon).

### Panel numbering example

```
┌───────┬───────┐
│  P0   │  P1   │  ← top row: left→right
├───────┼───────┤
│  P2   │  P3   │  ← bottom row: left→right
└───────┴───────┘
reading_order: [0, 1, 2, 3]
```

### Webtoon format

Many manhwa pages are tall vertical strips. Panels flow top to bottom with minimal horizontal separation. Each visual "beat" is a panel even if there are no borders.

### Within a panel: bubble reading order

Left to right, top to bottom — same as English text.

### Korean bubble conventions

- Speech bubbles have rounded shapes with tails, similar to western comics
- Narration: rectangular box, often at top of panel, different fill color
- Thought: cloud shape or italicized text in lighter bubble
- SFX: large stylized hangul or romanized sounds drawn into the art
  Common examples: 쾅 (KWANG — bang), 휙 (HWIK — whoosh), 하하 (HAHA — laughter)

### Typical manhwa narrative features

- More detailed backgrounds than manga
- Colored pages are standard (not black and white)
- Facial expressions and emotion lines are prominent visual storytelling tools
